# CMBL Profile Update Guide
## Adding Non-Licensed NIGP Codes to Your Texas Comptroller Profile

---

## What You're Doing and Why

The CMBL (Centralized Master Bidders List) is the Texas Comptroller's vendor database. State agencies search CMBL by NIGP commodity code to find vendors for spot purchases and solicitations.

Your current profile has 5 NIGP codes covering licensed trades (HVAC, electrical, general building maintenance).

**You need to add 5 new non-licensed codes** to get discovered for spot purchases that don't require trade licenses:

| Code | Service |
|---|---|
| 910-39 | Janitorial / Custodial Services |
| 968-94 | Pressure Washing Services |
| 910-54 | Painting and Wall Covering |
| 910-25 | Flooring Maintenance and Repair |
| 910-81 | Window Washing Services |

## Step-by-Step: Update Your CMBL Profile

### Step 1: Log In to Texas SmartBuy Vendor Portal

1. Go to **esbd.texas.gov**
2. Click **"Vendor Login"** or look for the SmartBuy vendor portal
3. Log in with your credentials (email and password you set up during registration)
4. If you haven't completed registration, you need to do that first (see Step 0 below)

### Step 0: If Not Yet Registered on Texas SmartBuy

1. Go to **esbd.texas.gov**
2. Click **"Register as a Vendor"**
3. Enter your B2G VID: **21829543** when prompted — this links your SmartBuy account to your existing Comptroller HUB certification
4. Complete the registration form with your entity information (Joe Sanches LLC)
5. Set your login credentials
6. Once registered, proceed to Step 2

### Step 2: Navigate to Your Commodity Code Profile

Once logged in:
1. Look for **"My Profile"** or **"Vendor Profile"**
2. Find the section labeled **"Commodity Codes"** or **"NIGP Codes"**
3. You should see your current codes listed:
   - 91000, 91082, 91141, 91450, 98852

### Step 3: Add New NIGP Codes

For each new code you want to add:

1. Click **"Add Commodity Code"** or the **"+"** button
2. In the search box, type the NIGP code number OR the keyword:
   - Type **"910-39"** or search **"janitorial"**
   - Type **"968-94"** or search **"pressure washing"**
   - Type **"910-54"** or search **"painting"**
   - Type **"910-25"** or search **"flooring"**
   - Type **"910-81"** or search **"window washing"**
3. Select the matching code from the results
4. Click **"Add"** or **"Save"**

Repeat for all 5 new codes.

### Step 4: Save and Confirm

1. After adding all codes, click **"Save"** or **"Submit"**
2. Your profile should now show 10 NIGP codes total
3. Take a screenshot for your records
